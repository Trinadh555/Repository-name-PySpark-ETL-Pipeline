# Sales Performance Dashboard (Power BI)

## Overview
Developed an interactive Power BI dashboard for analyzing sales performance, customer trends, and regional distribution.

## Features
- KPI Cards (Total Sales, Average Sales, Total Orders)
- Sales by Category
- Regional Sales Distribution
- Top 5 Customers by Sales
- Yearly Sales Trend
- Interactive Filters (Category & Region)

## Tools Used
- Power BI
- Power Query
- DAX
